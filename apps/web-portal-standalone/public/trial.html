Step by Step

เริ่มจากพื้นที่ก่อนเปิดใช้งาน แล้วค่อยเปิด runtime ที่จำเป็นจริง

แนวทางนี้ช่วยให้คุณไม่ต้องเปิดทุกระบบพร้อมกันตั้งแต่วันแรก เริ่มจากดูเมนูและสิทธิ์ก่อน แล้วค่อยตัดสินใจว่าจะใช้ Delivery Agent, Server Bot หรือทั้งสองอย่างตามงานจริงของชุมชน

สร้างบัญชี เปิดพื้นที่ก่อนใช้งาน
สิ่งที่ควรเช็กก่อนเปิดจริง

เห็นของจริงแบบล็อก action ไว้ก่อน

  • เปิดดูหน้า Server Status, Orders, Players, Delivery Agents และ Server Bots ได้ครบในพื้นที่ก่อนเปิดใช้งาน
  • หน้าที่ยังต้องใช้แพ็กเกจหรือ runtime เพิ่ม จะขึ้นสถานะล็อกและคำแนะนำชัดเจน
  • อัปเกรดภายหลังแล้วใช้โครงเดิมต่อได้ โดยไม่ต้องเริ่ม onboarding ใหม่
1. สร้างพื้นที่เริ่มต้น

เปิด tenant เริ่มต้นเพื่อดูเมนูและสิทธิ์

เริ่มจากหน้า signup เพื่อสร้างบัญชีและพื้นที่ทำงาน จากนั้นดูเมนูที่เปิดให้ใช้ได้จริงและรายการฟีเจอร์ที่ยังต้องอัปเกรดก่อนเปิด production

2. ตัดสินใจเรื่อง runtime

เลือกให้ตรงว่าต้องใช้ Delivery Agent หรือ Server Bot

ถ้าต้องการส่งของในเกม ใช้ Delivery Agent ถ้าต้องการ sync log แก้ config สำรองข้อมูล หรือควบคุมเซิร์ฟเวอร์ ใช้ Server Bot

3. เปิดให้ผู้เล่นใช้งาน

ค่อยเปิด Player Portal หลังจากฝั่งผู้ดูแลนิ่งแล้ว

เมื่อฝั่งผู้ดูแลพร้อมแล้ว จึงเปิด Player Portal เพื่อให้ผู้เล่นเข้าดูร้านค้า กระเป๋าเงิน คำสั่งซื้อ และกิจกรรมของชุมชนจากหน้าจริง
