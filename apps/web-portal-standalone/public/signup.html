สร้างบัญชีและพื้นที่ทำงาน

หลังสมัคร ระบบจะสร้างพื้นที่ใช้งานเริ่มต้นให้คุณทันที เพื่อให้ลองดูเมนู สิทธิ์ และโครงงานก่อนเปิดใช้งานจริง โดยไม่ต้องซื้อก่อน

หลังสร้างบัญชี

เห็นโครงงานจริง แต่ action สำคัญยังถูกควบคุมตามสิทธิ์

  • เปิดดูหน้า Dashboard, Server Status, Orders, Players, Delivery Agents และ Server Bots ได้จริง
  • ปุ่มที่ต้องใช้แพ็กเกจหรือ runtime เพิ่มจะขึ้นสถานะชัดเจนว่า “ต้องอัปเกรด” หรือ “ต้องเปิดเพิ่ม”
  • อัปเกรดภายหลังแล้วใช้พื้นที่เดิมต่อได้ ไม่ต้องเริ่ม onboarding ใหม่
