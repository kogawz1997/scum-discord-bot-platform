Packages

เลือกชุดฟีเจอร์ให้ตรงกับงานจริงของเซิร์ฟเวอร์

ทุกแพ็กเกจเริ่มจากการสร้างบัญชีและพื้นที่ทำงานได้ก่อน เพื่อให้คุณเปิดดูเมนู โครงงาน และขอบเขตสิทธิ์ที่ได้จริง แล้วค่อยเปิดใช้งานตามแผนของทีม

BOT_LOG

ซิงก์ log และ Discord integration

เหมาะกับชุมชนที่ต้องการ Server Bot สำหรับ log dashboard การเฝ้าดูเหตุการณ์ และการเชื่อม Discord เป็นหลัก

เริ่มต้นด้วยแพ็กเกจนี้
BOT_LOG_DELIVERY

เพิ่มร้านค้า คำสั่งซื้อ และการส่งของ

เหมาะกับเซิร์ฟเวอร์ที่เปิดขายสินค้าและต้องการ Delivery Agent ควบคู่กับ Server Bot เพื่อดูแล flow การส่งของครบชุด

เริ่มต้นด้วยแพ็กเกจนี้
FULL_OPTION

ชุดครบสำหรับโฮสต์และคอมเมิร์ซ

รวมการควบคุมเซิร์ฟเวอร์ การตั้งค่า สถานะรันไทม์ ร้านค้า คำสั่งซื้อ และพอร์ทัลผู้เล่นครบชุด เหมาะกับการเปิดระบบเต็มรูปแบบ

เริ่มต้นด้วยแพ็กเกจนี้
SERVER_ONLY

เน้นงานโฮสต์และควบคุมเซิร์ฟเวอร์

เหมาะกับงานดูแลเซิร์ฟเวอร์โดยไม่เปิด commerce หรือ player-facing delivery แต่ยังต้องการสถานะ ระบบควบคุม และการจัดการฝั่งผู้ดูแล
