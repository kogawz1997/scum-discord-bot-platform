Managed SCUM Platform

แยกบทบาทชัด ดูแลง่าย และพาแต่ละทีมไปทำงานในหน้าที่ถูกต้องตั้งแต่วันแรก

ระบบนี้แยก Owner Panel, Tenant Admin Panel และ Player Portal ออกจากกันชัดเจน พร้อมแยก Delivery Agent และ Server Bot ออกจากกันในระดับการใช้งานจริง เพื่อให้ทุกคนเริ่มงานได้ทันทีโดยไม่ต้องเดาว่าหน้าไหนใช้ทำอะไร

สร้างบัญชีและพื้นที่ทำงาน เปรียบเทียบแพ็กเกจ
หลายผู้เช่า ลองโครงระบบก่อนเปิดจริง แยก Delivery Agent กับ Server Bot
เริ่มต้นให้ถูกทาง

เจ้าของระบบ ผู้ดูแลผู้เช่า และผู้เล่น ใช้คนละพื้นที่ แต่ทำงานเชื่อมกันได้จริง

เจ้าของระบบใช้ดูภาพรวมแพลตฟอร์มและรายได้ ผู้ดูแลผู้เช่าใช้ดูแลงานประจำวันของเซิร์ฟเวอร์ ส่วนผู้เล่นใช้ติดตามร้านค้า กระเป๋าเงิน คำสั่งซื้อ สถิติ และกิจกรรมของชุมชนจากพอร์ทัลเฉพาะของตัวเอง

Owner Panel

Tenant Admin Panel

เปิดดูสถานะเซิร์ฟเวอร์ แก้ค่าระบบ ติดตามคำสั่งซื้อ ผู้เล่น การส่งของ Delivery Agent และ Server Bot ในหน้าที่ทีมดูแลใช้งานได้จริงทุกวัน

เข้า Tenant Panel
Player Portal

พอร์ทัลผู้เล่นที่ใช้ติดตามร้านค้าและชุมชนได้จริง

รวมร้านค้า กระเป๋าเงิน คำสั่งซื้อ การส่งของ สถิติ กิจกรรม และสถานะบัญชีไว้ในหน้าที่อ่านง่ายและกลับมาใช้งานซ้ำได้จริง

เข้า Player Portal
เริ่มต้นก่อนเปิดใช้งานจริง

สำรวจเมนู สิทธิ์ และสิ่งที่ต้องเปิดเพิ่มก่อนตัดสินใจ

หน้าพื้นที่ก่อนเปิดใช้งานช่วยให้คุณเห็นโครงเมนู สิทธิ์ และจุดที่ยังต้องอัปเกรดหรือเปิด runtime เพิ่มก่อนเริ่มใช้งานจริง โดยไม่ต้องสร้างระบบใหม่อีกรอบหลังตัดสินใจ

เปิดพื้นที่ก่อนเปิดใช้งาน ดูแผนเริ่มต้น
มองภาพรวมได้ทันที

เข้าใจบทบาทของแต่ละพื้นที่ก่อนเริ่ม onboard ทีมจริง

หน้าแนะนำระบบช่วยให้ทีมใหม่รู้ทันทีว่า Owner ใช้ทำอะไร Tenant ใช้ทำอะไร และผู้เล่นควรเข้าทางไหน โดยไม่ต้องเปิดหลายหน้ามาเดาเอง
