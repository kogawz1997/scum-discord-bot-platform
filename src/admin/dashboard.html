Owner Panel

ศูนย์ควบคุมแพลตฟอร์ม

ใช้สำหรับดูภาพรวมผู้เช่า แพ็กเกจ รายได้ สุขภาพรันไทม์ เหตุการณ์ และความปลอดภัยระดับแพลตฟอร์ม

  • จัดการ tenant, package และ subscription
  • ดู incident, audit และ runtime health
  • ควบคุม feature, diagnostics และ security ระดับระบบ

Tenant Admin Panel

แผงดูแลเซิร์ฟเวอร์

ใช้สำหรับงานประจำวันของแต่ละชุมชน เช่น สถานะเซิร์ฟเวอร์ คำสั่งซื้อ ผู้เล่น การส่งของ และการดูแล Delivery Agent กับ Server Bot แยกหน้ากันชัดเจน

  • ดูสถานะเซิร์ฟเวอร์และข้อมูลซิงก์ล่าสุด
  • ติดตามคำสั่งซื้อ การส่งของ และข้อมูลผู้เล่น
  • จัดการ Delivery Agent และ Server Bot คนละหน้า
