Role Separation

ทุกทีมมีพื้นที่ของตัวเอง และไม่ต้องเรียนรู้ทั้งระบบพร้อมกัน

หน้าแนะนำนี้สรุปให้ชัดว่าใครควรเข้าหน้าไหน Owner ใช้คุมแพลตฟอร์ม Tenant ใช้ดูแลงานประจำวัน และ Player ใช้เข้าร้านค้า คำสั่งซื้อ และข้อมูลชุมชนของตัวเอง

Runtime Separation

Delivery Agent กับ Server Bot เป็นคนละบทบาท และควรตั้งค่าแยกกัน

Delivery Agent ใช้กับการส่งของและการทำงานในเกม ส่วน Server Bot ใช้กับการซิงก์ log การแก้ config การสำรองข้อมูล และการควบคุมเซิร์ฟเวอร์ จึงต้องมองเป็นคนละระบบตั้งแต่ตอนวางแผน
